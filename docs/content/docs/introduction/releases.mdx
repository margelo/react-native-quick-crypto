Stay up to date with the latest changes in RNQC.

## Compatibility Matrix

| Version | RN Architecture | Modules |
| ------- | ------ | ------- |
| `1.x`     | **New Architecture** [->](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/enable-apps.md)  | **Nitro Modules** [->](https://github.com/mrousavy/nitro) |
| `0.x`     | Old (Bridge) & New (Interop)  | Bridge & JSI |

> [!NOTE]
> **Version 1.x** is a major refactor (Nitro Modules). It requires **React Native 0.75+**.
> If you need to support earlier versions, please continue using `0.x`.
