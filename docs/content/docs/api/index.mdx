RNQC mirrors the Node.js `crypto` API while adding specialized high-performance modules.

## Core Modules

<Cards>
  <Card title="install()" href="/docs/api/install">
    Global polyfill injection for drop-in compatibility.
  </Card>
  <Card title="Cipher" href="/docs/api/cipher">
    Symmetric encryption (AES, ChaCha20).
  </Card>
  <Card title="Hash" href="/docs/api/hash">
    Message digests (SHA-2, SHA-3, MD5).
  </Card>
  <Card title="HMAC" href="/docs/api/hmac">
    Keyed-hash message authentication.
  </Card>
   <Card title="Random" href="/docs/api/random">

  </Card>
  <Card title="Keys" href="/docs/api/keys">

  </Card>
</Cards>
